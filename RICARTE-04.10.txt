Além disso, o grupo decidiu por criar mais uma nova classe, a classe Aluguel. Essa nova classe terá como atributos a quantidade de dias pelos quais o filme
foi alugado e qual o filme que foi alugado. O atributo que diz respeito a qual filme foi alugado será do tipo Filme (classe Filme) e o atributo que diz por
quantos dias o filme foi alugado será do tipo int. O construtor, getters e setters dessa classe também foram implementados.  
Quanto maior o número de dias pelos quais o filme é alugado, mais pontos o cliente ganha com o locadora (pontoAdquiridos). O código da classe segue abaixo:

package locadora;

/**
 *
 * @author Otavio
 */
public class Aluguel {
    private int diasAlugado;
    private Filme filme;
    
    public Aluguel(int diasAlugado, Filme filme){
            this.diasAlugado = diasAlugado;
            this.filme = filme;
    }
    
    public int getDiasAlugado() {
        return diasAlugado;
    }
    
    public void setDiasAlugado(int diasAlugado) {
        this.diasAlugado = diasAlugado; 
    }
    
    public Filme getFilme() {
        return filme;
    }
    
    public void setFilme(Filme filme) {
        this.filme = filme;
    }
}
